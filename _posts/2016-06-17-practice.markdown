# Linux 를 이용한 github 이용법

1. git init
..* 현재 폴더를 git 로컬저장소로 설정
2. git clone chyeong@chyeonghoi.github.io
..* 현재 폴더에 원격저장소 복사
3. git add 
..* 로컬 저장소1에 저장
4. git commit -m "add"
..* 로컬 저장소2에 저장
5. git push origin master
..* 원격 저장소에 push
